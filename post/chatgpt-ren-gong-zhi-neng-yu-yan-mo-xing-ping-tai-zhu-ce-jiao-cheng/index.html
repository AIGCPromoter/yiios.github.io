ChatGPT 人工智能语言模型平台注册教程

2023-02-21 2 min read # ChatGPT # OpenAI

大致的步骤如下：

  1. 需要确保你可以在扶墙状态下访问 openai.com。不要使用香港，最好使用美国，建议注册过程挂全局，并且使用浏览器的隐私模式提高成功率。注册成功即可在非隐私模式下登录账号，建议使用 ChatGPT 时全程保持扶墙状态。
  2. 打开后面网址进行注册https://beta.openai.com/signup。
  3. 在注册过程中会进行手机短信验证。你可以使用获取验证码平台，在该平台注册后充值几元左右（支持支付宝）即可。接着在该平台左侧搜索OpenAI，选择一个结果后，点击购物车(适用于sms-activate.org)。
  4. 在 openai 的验证页面上，填写上一步获取的手机号码（注意选择正确的国家码）。
  5. 回到获取验证码平台，不久后应该会收到验证码，将其填入openai的页面中。
  6. 注册完成后，你可以在 ChatGPT 网站上进行登录并开始使用。https://chat.openai.com/auth/login

验证码平台

  • onlinesim.io(接码费用0.25美元，首冲1.28美元)
  • sms24.info(免费接短信，需要多试试，免费的都不会有试用金，有官网条件的可用)
  • 2NR(免费接码应用，安卓可用，免费的都不会有试用金，有官网条件的可用)
  • sms-activate.org(首冲2美金，巨贵，还容易收不到，不太推荐)

自用 VPS 无法访问 OpenAI 的解决方案

使用 Warp 接管和 ChatGPT 有关的流量，或者接管全部流量。
WARP 脚本：https://github.com/P3TERX/warp.sh

购买账号(适合尝鲜或无扶墙工具用户)

https://shop.yiios.com/
如果没有机场，也可以购买账号后，使用购买获得的 api key在 https://ai.yiios.com/ 体验。
