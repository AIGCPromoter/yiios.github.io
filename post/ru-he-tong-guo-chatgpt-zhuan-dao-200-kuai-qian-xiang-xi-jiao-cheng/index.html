如何通过 ChatGPT 赚到 200 块钱 - 详细教程

2023-03-13 1 min read
  1. 使用开源代码搭建一个镜像站点.
  2. 对不使用自有 key 的用户进行适当的限制.
  3. 卖 api key,2.4 元一个的进价,6 元出售,毛利润 60%.

目前镜像站运营 20 天,一共 5125 uv / 8000 pv,卖出 64 个账号,用户转化率 1.24%. 扣除 10 块钱给免费用户的提供 key 费用.净盈利 220 块钱.

我的站点只限制了单次对话次数不超过 5 次。这样做有两个好处：

  1. 可以牟利
  2. 防止上下文过大导致 token 消耗过快
