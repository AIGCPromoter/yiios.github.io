小鱼镜像站使用教程

购买

密钥购买地址：https://s.yiios.com/

按需购买API的密钥，付款后密钥信息将发送至你下单时填写的邮箱中。

使用教程

镜像站地址：https://ai.yiios.com/

1. 访问 ai.yiios.com

2. 填入 API 密钥

2.1 点击左下角的设置按钮进入设置页

2.2 找到配置项： API 密钥，填入收到的密钥

往下翻设置页，找到接口地址和API密钥(必填)两项。

填入API密钥(必填)，购买后你会收到发货卡密邮件：

4.0卡密：

您购买的卡密如下：密钥yiios-UBYPH98R0M7EXlVSJT3BlbkFJ888482399BdE80a1e7eCe38使用说明https://www.yiios.com/post/gpt-4-api-shi-yong-shuo-ming/

示例中API密钥为：yiios-UBYPH98R0M7EXlVSJT3BlbkFJ888482399BdE80a1e7eCe38，结合你收到的卡密信息，将收到的密钥填入API密钥(必填)设置项中即可。

3. 开始对话
